# niscript

ナイスなシェルを集めたniscriptです。

## ナイスなシェルたち

1. rpl_filename.sh
    
    あるディレクトリ下の特定の文字列を含むファイル名を置換する。(今の所直下のファイルのみ・多分ディレクトリは無理）


2. rpl_text.sh
    
    あるディレクトリ下のファイルに含まれる特定の文字を置換する。(今の所直下のファイルのみ・多分ディレクトリは無理）
